# SQL Project: Shop Management System

This project contains SQL queries and scripts demonstrating the management of a shop database. It includes:

- Creating tables and inserting data
- Writing SELECT queries for analysis
- Using JOIN, GROUP BY, and aggregate functions
- Example outputs (see screenshots folder)

## Purpose
This project was completed as part of a university module to demonstrate SQL skills.  
It includes creating and managing a shop database, writing queries for data analysis, insert bulk data, and applying SQL concepts learned during the course.
